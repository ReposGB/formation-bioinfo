Introduction à Python

Python est un langage de haut niveau interprété initialement publié en 1991. Il a progressivement pris le pas sur Perl en bioinformatique jusqu’à le dépasser fin des années 2010.

Depuis, les versions se sont succédées, Python est maintenant utilisé sur un grand nombre de plateformes et pour des applications très diversifiées.

Aujourd’hui, la dernière version disponible est 3.9.5. 2.7 n’est plus supporté depuis 2020. (https://www.python.org/downloads/)

C’est un langage open-source. avec une importante communauté de développeurs.

Dans le domaine de la bioinformatique, Python est plutôt utilisé dans le développement d’applications et de pipelines (exemple: transformations de fichiers).

Charactéristiques

Python possède:

  • Une syntaxe claire et cohérente
  • Une indentation significative: Ce sont les indentations qui structurent le code.
  • Une gestion automatique de la mémoire (Garbage collector)
  • Un Typage dynamique fort sans déclaration explicite.
  • On peut passer en développement orienté objet (pas obligatoire)
  • Structure multi-fichiers des applications, ce qui facilite le développement et la maintenance.
  • Un grand nombre de bibliothèques (packages) sont disponibles dans un grande variété de domaines.

Monde de fonctionnement

Python peut être utilisé en mode interactif, ou comme un language interprété rapide.

  • Les commandes peuvent être enregistrées dans un fichier .py appelé script. Python peut ensuite directement executer ce fichier (Il crée un bytecode intermédiaire)

  • On peut utiliser un interpréteur (Exemple idle, fourni avec python) pour obtenir un résultat immédiat aux commandes (comme avec l’interpreteur R)

Style de nommage

Durant ce cours, nous allons tenter de respecter les règles de nommage suivantes:

  • UPPERCASE ou UPPER_CASE pour les constantes
  • TitleCase pour les classes
  • UneExceptionError pour les exceptions
  • camelCase pour les fonctions, les méthodes et les interfaces graphiques
  • unmodule_m pour les modules
  • lowercase ou lower_case pour les variables

Enfin, on utilisera les notations suivantes pour faire des réservations

 _xxx # usage interne
__xxx # attribut de classe
__xxx__ # nom spécial réservé

Contrôle de flux d’instruction

Il existe 3 concepts importants à utiliser lorsque l’on structure un programme Python:

  • Les Blocs
  • Les Structures de contrôle:
    • Les Branchements
    • Les Boucles

Dans la syntaxe python, ce qui suit un ‘#’ est un commentaire. Les Blocs sont alignés au même niveau d’indentation, et les structures de contrôle sont délimitées par deux points ‘:’.

Les Blocs

Un Bloc d’instruction est une unité de programmation qui regroupe une séquence ordonnée d’instructions qui s’exécutent les unes après les autres.

Ces blocs sont branchés les uns aux autres à travers les structures de contrôle.

# Commentaire
instruction(bloc1)
instruction(bloc1)
structure_de_controle:
  instruction(bloc2)
  instruction(bloc2)

Exemple:

# Commentaire
i = 0
maxi =10
while (i < j):
  print("i est inférieur à maxi")
  i = i + 1

Les branchements

Les tests permettent de choisir un chemin d’exécution en fonction de la valeur de certaines variables.

if (condition1):
  instructions1
elif (copndition2):
  instructions2
else:
  instructions3

Exemple:

if (i < 5):
  print("i est inférieur à 5")
elif (i == 5):
  print("i est égal à 5")
else:
  print("i est supérieur à 5")

Les boucles for

for est un itérateur sur une collection d’objet. Il permet de parcourir les éléments de la collection.

for variable in collection:
  instructions

Exemples:

for i in range(5):
  print("i=", i)
## i= 0
## i= 1
## i= 2
## i= 3
## i= 4
for ch in "Hello":
  print("ch=", ch)

## ch= e
## ch= l
## ch= l
## ch= o

Les boucles while

Les boucles while permettent d’exécuter un bloc tant qu’un test est vrai. Il ne s’agit pas d’explorer l’intégralité des itérations mais de cherche quelque chose ou de faire une boucle infinie.

while (test):
  instructions

Exemple:

i = 0
while (i < 5):
  print("i est inférieur à 5")
  i = i + 1

Instructions de contrôle

Les mots clés suivants sont utilisables dans les boucles for et while:

  • break permet de sortir de la boucle la plus interne où l’on se situe
  • continue permet de passer directement au tour de la boucle suivants sans executer les instructions
  • else permet de rajouter des instructions qui seront exécutées si aucun break n’a été déclenché.

les conteneurs standard

En python, les conteneurs standards sont

  • Les listes
  • Les tuples
  • Les ensembles (sets)
  • Les dictionnaires

Les tuples

Les tuples permettent de regrouper des éléments de différentes nature en une seule structure. Ils ne sont pas modifiables. Il sont notés avec des parenthèses et des virgules pour séparer les éléments:

mon_tuple = ( 1, 2, 'abc')

Les opérateurs des tuples permettent la concaténation, la filtration ou l’extraction d’éléments, et sont très similaires à ceux des chaînes de caractères

len(t)     # nombre d'éléments dans le tuple
t1 + t2    # Concaténation de ’t1’ et ’t2’
t * n      # Répétition ’n’ fois du tuple ’t’
t[n]       # Extraction du n-ième élément du tuple en partant de la gauche
t[-n]      # Extraction du n-ième élément du tuple en partant de la droite
t[:n]      # Extraction des ’n’ premiers éléments du tuple
t[n:]      # Extraction des éléments ’n’ à la fin du tuple 
t[-n:]     # Extraction des ’n’ derniers éléments du tuple
t[n:m]     # Extraction des éléments qui sont entre ’n’ et ’m’

Les listes

Les listes servent à stocker des collections ordonnées d’éléments avec possibilité de les étendes ou de les manipuler facilement. On les note avec des crochets:

l = [6, 8, 24, 12 ]

Les opérateurs des listes permettent la concaténation, la filtration ou l’extraction d’éléments, et sont très similaires à ceux des chaînes de caractères.

len(l)       # Nombre d’éléments dans la liste
del(l[n])    # Détruit l’élément ’n’ de la liste ’l’
l.clear()    # Vide la liste ’l’
l.copy()     # Renvoie une copie de la liste ’l’
l.append(e)  # Ajoute l’élément ’e’ à la fin de la liste ’l’
l.insert(n, e) # Insère l’élément ’e’ à la position’n’ dans la liste ’l’ 
l1 + l2     # Concaténation de ’l1’ et ’l2’
l * n       # Répétition ’n’ fois de la liste ’l’
l[n]        # Extraction du n-ième élément en partant du début de la liste
l[-n]       # Extraction du n-ième élément en partant de la fin de la liste 
l[:n]       # Extraction des ’n’ premiers éléments de la liste
l[n:]       # Extraction des éléments ’n’ à la fin de la liste
l[-n:]      # Extraction des ’n’ derniers éléments de la liste
l[n:m]      # Extraction des éléments qui sont entre ’n’ et ’m’

Les ensembles

Les ensembles sont des listes non ordonnées d’éléments uniques (set en anglais), ils sont extensibles et ne gardent aucune notion d’ordre entre les éléments qu’ils contiennent. On les note avec des accolades:

s = { 'python', 'C++', 'javascript' }

Les opérations que l’on peut faire incluent les opérations mathématiques de type ensembliste:

len(s)       # Taille de l’ensemble ’s’
s.add(e)     # Ajoute ’e’ à ’s’
s.remove(e)  # Retire ’e’ de ’s’ (erreur si ’e’ ̸∈ ’s’)
s.discard(e) # Retire ’e’ de ’s’
s.clear()    # Vide l’ensemble ’s’
s1 | s2      # Union de ’s1’ et ’s2’
s1 & s2      # Intersection de ’s1’ et ’s2’ 
s1 - s2      # Différence de ’s1’ et ’s2’
s1 ^ s2      # Différence exclusive de ’s1’ et ’s2’

Les ensembles (suite)

Les opérations que l’on peut faire incluent les opérations mathématiques de type ensembliste: (suite)

e in s        # ’e’ ∈ ’s’
e not in s    # ’e’ ̸∈ ’s’
s1 == s2      # ’s1’ égal ’s2’
s1 != s2      # ’s1’ différent de ’s2’
s1 < s2       # ’s1’ est inclus (strict) dans ’s2’ 
s1 <= s2      # ’s1’ est inclus ou égal à ’s2’
s1 > s2       # ’s2’ inclus (strict) dans ’s1’ 
s1>=s2        # ’s2’ inclus ou égal à ’s1’
s1.isdisjoint(s2)  # ’s1’ disjoint de ’s2’
s1.issubset(s2)    # ’s1’ inclus dans ’s2’
s1.issuperset(s2)  #  ’s1’ inclus ’s2’

Les dictionnaires (ou tableaux associatifs)

Les dictionnaires sont des ensembles associatifs associant une clé en entrée avec une valeur. Les dictionnaires sont encadrés par des accolades et chaque couple ’clef:valeur’ est séparé par une virgule:

mon_dict = {  'mardi': 2, 'dimanche':7 }

Et on accède à la valeur comme suit: mon_dict['mardi'].

Les opérateur permettent de faire des recherche, d’ajouter et modifier des entrées:

len(d)     # Nombre d’éléments du dictionnaire
del(d[k])  #  Détruit l’entrée ’k’ du dictionnaire
d.clear()  # Vide le dictionnaire ’d’
d.copy()   # Renvoie une copie du dictionnaire ’d’
d[k]       # Accède à la valeur de la clef ’k’
d[k] = v   # Assigne la valeur ’v’ à la clef ’k’
k in d     # Teste si la clef ’k’ est présente dans le dictionnaire ’d’
k not in d # Teste si la clef ’k’ n’est pas présente dans le dictionnaire ’d’

Les fonctions

Les fonctions permettent de rendre un programme modulaire en évitant de réécrire plusieurs fois le même code. Elle est définie par son prototype comportant le mot-clé def, son nom, ses arguments et aussi son type de retour.

# Défninition d'une fonction
def ma_fonc(args):
  instructions
  return(x)
  
# Programme principal
instructions
x = ma_fonc(args)
instructions
# Défninition d'une fonction
def multi(a, b):
  res = a * b
  return(res)
  
# Programme principal
a = 2; b = 4
res_multi = multi(a, b)
print(res_multi)

Les arguments sont les variables passées à la foinction. Ils ne sont pas typés a priori mais substitués dynamiquement dans le code lors de l’execution. Du coup, il est possible d’appliquer la fonction à différents type sans avoir à la réécrire.

Le type de retour correspond à ce que retourne la fonction. Le mot clé return permet de définir la fin de la fonction et de définir ce qui est retourné. On peut retourner plusieurs variables en Python, ou None.

Passage par valeur

Dans la plupart des langages, il existe deux modes de passage des arguments. Lors du passage des arguments par valeur, seule une copie des valeurs sont utilisées, mais pas les variables originales. Les changements effectués sur les arguments sont locaux à la fonction et le seul moyen de récupérer les changements est de passer par l’instruction return.

Exemple de passage d’argument par valeur:

def affi(s):
  s = "Je suis dans la fonction !"
  print (s)

s = "Je suis dans le main !"
affi(s)
## Je suis dans la fonction !
print(s)
## Je suis dans le main !
def affi(s):
  s = "Je suis dans la fonction !"
  print (s)
  return s
s = "Je suis dans le main !"
s= affi(s)
## Je suis dans la fonction !
print(s)
## Je suis dans la fonction !

Passage par référence

Dans la plupart des langages, il existe deux modes de passage des arguments. Lors du passage des arguments par valeur, seule une copie des valeurs sont utilisées, mais pas les variables originales. Les changements effectués sur les arguments sont locaux à la fonction et le seul moyen de récupérer les changements est de passer par l’instruction return.

Exemple de passage d’argument par référence:

Contrairement à l’exemple précédent, d est modifié même sans la présence du return.

def modif(d):
  d['cle'] = "Je suis dans la fonction !"
  # pas de return
  
d = { 'cle': "Je suis dans le main !", } 

modif(d)
print(d)
## {'cle': 'Je suis dans la fonction !'}

Variables locales/globales

La portée d’une variable (variable scope) correspond aux zone du programmes dans lesquelles cette variable peut être appelée.

Une variable est dite globale si sa portée recouvre l’ensemble du programme. Une variable est dite locale si sa portée est limitée à une partie du programme, par exemple à une fonction.

def affi():

  s = "Je suis dans la fonction !"
  print (s)

s = "Je suis dans le main !"
affi()
## Je suis dans la fonction !
print(s)
## Je suis dans le main !
def affi():
  global s
  s = "Je suis dans la fonction !"
  print (s)

s = "Je suis dans le main !"
affi()
## Je suis dans la fonction !
print(s)
## Je suis dans la fonction !

Documenter ses fonction

Les docstrings sont des chaînes de caractères qui suivent immédiatement le def et qui constituent la documentation de la fonction en question. Cette documentation permet de comprendre ce que fait la fonction et d’aider à l’utiliser correctement.

def multi(a, b):
  """This function multiplies its two arguments.
  Exemple: res = multi(10,11)."""
  res = a * b
  return(res)
help(multi)
## Help on function multi in module __main__:
## 
## multi(a, b)
##     This function multiplies its two arguments.
##     Exemple: res = multi(10,11).

Utilitaires: enumerate et zip

Les fonctions enumerate() et zip() sont utiles pour les itérations sur des structures.

  • enumerate: Associe un index sur une structure lorsqu’il n’y en a pas

  • zip: Permet d’itérer deux structures en parallèle, même de tailles différentes

my_set = { 'a', 'b', 'c', 'd'}

for index, item in enumerate(my_set):

  print(index, item)
## 0 b
## 1 c
## 2 d
## 3 a
mylist1 = [5, 3, 9, 4]
mylist2 = ['a', 'w', 'd']
for index, char in zip(mylist1, mylist2):
  print (index , char)
## 5 a
## 3 w
## 9 d
